# Dreambox App 🚀 (Work In Progress)

Dreambox is an immersive and interactive Visual Novel (VN) player application built using Flutter and Supabase. The app allows users to experience rich storytelling with dynamic choices, complete sound management, and an efficient database-driven asset architecture.

> **Note for Recruiters:** This project is actively under development as part of my practical learning journey to master scalable application architecture and database management.

---

## 🌟 Key Features

*   **Interactive Visual Novel Engine:** A custom-built player framework to handle complex story-driven logic.
*   **Structured Architecture:** Clean and scannable code separated into multiple operational layers (`HomeScreen`, `Library`, and `Profile`).
*   **Supabase Backend Integration:** Real-time database management handling story metadata, user progress tracking, and secure asset indexing.
*   **JSON-Based Episode Configuration:** High-performance, low-lag system layout to parse multi-branched storyline scripts dynamically from single-row database inputs.
*   **Asset & Audio Synchronization:** Optimized asset streaming pipeline ensuring zero user-facing lag for high-fidelity audio, dynamic background music, and layered character sprites.

## 🚀 How It Works (Overview)

The application workflow transitions from a curated **HomeScreen** dashboard to a detailed **Story Detail Screen** containing extensive structural metadata and an organized episode timeline. From there, users seamlessly launch the robust **VN Player Screen** for immersive runtime episode playback.
